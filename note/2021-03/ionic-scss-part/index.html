Ionic SCSS with Part
  • Mar 8, 2021
  • One minute read
  1. 利用::part(container)來選出part=“container"的元素，設定其基本樣式。
  2. 當.radio-checked出現的時候，改變背景顏色
  ion-radio {

    &.radio-checked::part(container) {
      background: var(--ion-color-primary);
    }

    &::part(container) {
      height: 20px;
      width: 20px;
      border: 1px solid #cccccc;
      border-radius: 10px;
    }
  }
